# Data Folder

This folder contains information about the dataset used in this project.

## Dataset Used
This project uses the **UNSW-NB15** dataset for network intrusion detection.

## Files Used
- `UNSW_NB15_training-set.csv` — training dataset with **175,341** records
- `UNSW_NB15_testing-set.csv` — testing dataset with **82,332** records
- `NUSW-NB15_features.csv` — feature description file used for understanding the dataset columns

## Source
- Official UNSW-NB15 dataset page: https://research.unsw.edu.au/projects/unsw-nb15-dataset
- Kaggle mirror: https://www.kaggle.com/datasets/alextamboli/unsw-nb15

## Note
The raw dataset files are **not uploaded to this repository** because of their size.  
This repository includes only project code, notebooks, and documentation.
